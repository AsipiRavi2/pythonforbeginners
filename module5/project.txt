Make a banking application using Python with below functionalities.

1.	user will be prompted with a menu.
	menu will contain below options and based upon user input, transaction would take place
	1. Check Balance
	2. Add Balance
	3. Withdraw Balance
	4. Exit
2. 	user can check balance.

3.	user can add balance.
	a. no negative number is accepted.
	
4.	user can withdraw balance.
	a. withdrawn only if withdraw balance is <= available balance.
	b. no negative number is accepted.

5.	user can do multiple transactions.
